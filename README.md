# Todo App

A simple and responsive Todo App built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Prevent empty tasks

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📁 Project Structure
- index.html
- style.css
- script.js
